This project is an educational project to teach my duaghter about using coding agents in software development. We will be making a very simple game, about something she finds very exciting, to help her learn about what AI can do and how it can help.


## The game's theme

The game is themed around her favourite movie: K-Pop demon hunters. This is a purely educational project and I want her to be excited about it --- it will never be released in public.

As a first step, we will implement a typical character creator tool, where we can choose the face of our K-Pop demon hunter hero. It should allow the player to select different components, like eyes, hair, mouth shape, face shape, etc. It should follow the inspired style of K-Pop demon hunters.
